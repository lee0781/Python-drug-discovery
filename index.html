📅 Day 2 — Tuesday

Mean, SD, and %CV

⏱ 30–40 min 🆕 np.mean(), np.std(), if/else
🔬 Scenario
Your lab ran a CellTiter-Glo assay on MCF-7 breast cancer cells. You have triplicates for both your DMSO control and drug-treated group. Calculate QC statistics before reporting results.

💡 The Concept

%CV = (Standard Deviation / Mean) × 100

This is the core QC metric for every assay. Below 10% = acceptable for cell-based assays.

💻 Code — day2_statistics.py

import numpy as np
np.set_printoptions(suppress=True)

# MCF-7 breast cancer cells — CellTiter-Glo RLU values
dmso_control = np.array([45320.0, 44870.0, 45110.0])
drug_treated = np.array([8920.0,  9210.0,  8750.0])

dmso_mean = np.mean(dmso_control)
dmso_std  = np.std(dmso_control, ddof=1)  # ALWAYS ddof=1
dmso_cv   = (dmso_std / dmso_mean) * 100

drug_mean = np.mean(drug_treated)
viability = (drug_mean / dmso_mean) * 100

print(f"DMSO Mean: {dmso_mean:.1f}  SD: {dmso_std:.1f}  %CV: {dmso_cv:.2f}%")
print(f"Viability: {viability:.1f}%")

if dmso_cv < 10:
    print("QC Status: PASS")
else:
    print("QC Status: FAIL — replate recommended")

✅ Expected Output

DMSO Mean: 45100.0 SD: 226.5 %CV: 0.50% Viability: 19.9% QC Status: PASS
⚡ Mini Challenge

📅 Day 5 — Monday

axis + for loops: Per-Row Statistics

⏱ 35–45 min 🆕 axis=1/0, for i in range(), elif
🔬 Scenario
You need a per-row summary of your plate — mean viability and a QC flag for each row. This is the standard format for plate QC reports in pharma.

💡 axis=1 vs axis=0

plate = [[96.0, 98.0, 94.0], row 0 [44.0, 46.0, 42.0], row 1 [ 4.0, 6.0, 2.0]] row 2 axis=1 → calculate ACROSS each row (→ → →) row 0 mean = (96+98+94)/3 = 96.0 ← one number per ROW row 1 mean = (44+46+42)/3 = 44.0 row 2 mean = ( 4+ 6+ 2)/3 = 4.0 result: [96.0, 44.0, 4.0] shape: (3,) axis=0 → calculate DOWN each column (↓ ↓ ↓) col 0 mean = (96+44+4)/3 = 48.0 ← one number per COLUMN result: [48.0, 50.0, 46.0] shape: (3,)

💡 The for Loop

for i in range(3): print(i) → i=0 → prints 0 → i=1 → prints 1 → i=2 → prints 2 range(3) generates: 0, 1, 2 (stops BEFORE 3)

📅 Day 6A — Wednesday

Boolean Masking: The Basics

⏱ 25–30 min 🆕 >, <, ==, array[ mask ]
🔬 Scenario
Your lab measured cell survival after drug treatment. You want to automatically flag wells below 20% — these are your potential drug hits.

💡 The Concept

wells = [98.2, 95.7, 12.3, 11.8, 97.1, 8.4] wells < 20: [False, False, True, True, False, True] ↑ ↑ ↑ only these wells pass the filter wells[mask]: [12.3, 11.8, 8.4] ← only True positions kept Step 1: Create mask mask = wells < 20 Step 2: Apply mask hits = wells[mask] Step 3: Use result print(hits)

💻 Code — day6a_masking.py

import numpy as np
np.set_printoptions(suppress=True)

wells = np.array([98.2, 95.7, 12.3, 11.8, 97.1, 8.4])

# Step 1: Create mask
mask = wells < 20
print("Mask:", mask)

# Step 2: Apply mask
hit_wells = wells[mask]
print("Hit wells:", hit_wells)

# Step 3: Count
print(f"Total: {wells.size}  |  Hits: {hit_wells.size}")

📅 Day 6B — Thursday

Combining Conditions: & and |

⏱ 30–35 min 🆕 & (AND), | (OR), np.sum()
🔬 Scenario
You want to flag drug hits that are below 30% viability AND not obvious measurement errors (which show up above 105%). One condition alone is not enough.

💡 The Concept

wells = [98.2, 12.3, 110.5, 8.4, 45.2, 3.1] condition 1: wells < 30 [False, True, False, True, False, True] condition 2: wells < 105 [ True, True, False, True, True, True] combined: (wells < 30) & (wells < 105) [False, True, False, True, False, True] ↑ ↑ ↑ BOTH conditions must be True to keep the well & = AND — both must be True | = OR — either can be True ⚠️ Always wrap each condition in parentheses! ✅ (wells < 30) & (wells < 105) ❌ wells < 30 & wells < 105 ← operator order bug

💻 Code — day6b_conditions.py

import numpy as np
np.set_printoptions(suppress=True)

wells = np.array([98.2, 12.3, 110.5, 8.4, 45.2, 3.1,
                  95.7, 22.1,  88.3, 0.2, 67.4, 14.8])

# AND: hits that are not errors
hit_mask  = (wells < 30) & (wells < 105)
hit_wells = wells[hit_mask]
print("Hits (< 30% AND < 105%):", hit_wells)

# OR: extreme wells (very high OR very low)
extreme_mask  = (wells > 100) | (wells < 5)
extreme_wells = wells[extreme_mask]
print("Extreme wells:", extreme_wells)

# Count Trues using np.sum()
print("Hit count:", np.sum(hit_mask))
# np.sum() on a boolean: True=1, False=0 → gives count

📅 Day 6C — Friday

np.abs() and ~ (NOT)

⏱ 25–30 min 🆕 np.abs(), ~ (invert mask)
🔬 Scenario
Some wells in your screen deviate far from the plate average — either too high or too low. You want to flag these outliers regardless of direction, then remove them to get clean data.

💡 np.abs() — Absolute Value

z_scores = [-2.8, 0.4, 1.2, -3.1, 0.9] np.abs(z_scores): [2.8, 0.4, 1.2, 3.1, 0.9] ↑ ↑ negative signs removed — now we can use > threshold to catch BOTH high and low outliers with one line

💡 ~ (NOT) — Inverting a Mask

mask = [False, False, True, True, False] ~mask = [ True, True, False, False, True] Use ~ to get "everything except": clean = wells[~outlier_mask] ← keep NON-outliers

💻 Code — day6c_abs_not.py

import numpy as np
np.set_printoptions(suppress=True)

deviations = np.array([-2.8, 0.4, 1.2, -3.1, 0.9, -0.3, 2.1, -1.7])

# Absolute value — removes negative signs
abs_dev = np.abs(deviations)
print("Absolute:", abs_dev)

# Flag outliers: |deviation| > 2.5
outlier_mask = np.abs(deviations) > 2.5
print("Outliers:", deviations[outlier_mask])

# Keep clean wells using ~
clean_wells = deviations[~outlier_mask]
print("Clean:", clean_wells)

print(f"Outliers: {np.sum(outlier_mask)}  |  Clean: {np.sum(~outlier_mask)}")

✅ Expected Output

Absolute: [2.8 0.4 1.2 3.1 0.9 0.3 2.1 1.7] Outliers: [-2.8 -3.1] Clean: [ 0.4 1.2 0.9 -0.3 2.1 -1.7] Outliers: 2 | Clean: 6

🔬 Scenario
Your lab treated cancer cells with increasing concentrations of a drug and measured how many cells survived. The relationship between drug dose and cell death follows an S-shaped curve. Today you'll load that data and find the approximate IC50 — the concentration that kills 50% of cells.

💡 Stage 1 Recap

Before starting Stage 2, answer these from memory:

1. How do you create a NumPy array? 2. What does plate[0, :] return? 3. What does np.mean(plate, axis=1) give you? 4. Difference between & and 'and' for arrays? 5. What does np.abs() do?

💡 What Is a Dose-Response Curve?

Low dose → most cells survive → high viability High dose → most cells die → low viability Viability 100% |●● | ●● 50% |──────●●●── ← IC50 threshold | ●●● 0% |________________●●● 0.1 1 10 100 1000 nM Concentration (log scale) IC50 = concentration that kills 50% of cells

💡 Why Log Scale?

Concentrations: 0.1, 1, 10, 100, 1000 nM Linear scale: |0.1--1-----------10-----------100-----------1000| ↑ cramped Log scale: |0.1---1---10---100---1000| ↑ evenly spaced, S-curve visible np.log10(10) = 1 np.log10(100) = 2 np.log10(1000) = 3

💻 Code — day1_dose_response_intro.py

import numpy as np
np.set_printoptions(suppress=True)

# Doxorubicin dose-response in 4T1 cells
concentrations = np.array([0.1, 0.3, 1.0, 3.0, 10.0, 30.0, 100.0])
viability      = np.array([98.2, 95.4, 87.3, 71.2, 48.6, 18.3, 4.2])

# Log transform concentrations
log_conc = np.log10(concentrations)
# np.log10() = log base 10 of every element
print("Log10 concentrations:", np.round(log_conc, 2))

# Find approximate IC50
diff_from_50 = np.abs(viability - 50)
# np.abs() — from Stage 1 Day 6C
ic50_index   = np.argmin(diff_from_50)
# np.argmin() returns the INDEX of the smallest value
# smallest distance from 50% = closest to IC50

print(f"Approximate IC50: {concentrations[ic50_index]:.1f} nM")
print(f"Viability at IC50: {viability[ic50_index]:.1f}%")

📅 Stage 2 — Day 3 — Wednesday

IC50 Calculation with np.interp()

⏱ 35–45 min 🆕 np.interp(), [::-1], 10**x
🔬 Scenario
Your PI wants the exact IC50 value — not just the closest data point. Today you'll use interpolation to calculate it precisely, and mark it on your dose-response plot.

💡 What Is Interpolation?

Known points: conc=3nM → viability=71.2% conc=10nM → viability=48.6% Where does viability = exactly 50%? → Somewhere between 3 and 10 nM → np.interp() finds the precise crossing point → Answer: ≈ 9.2 nM

💡 [::-1] — Reversing an Array

viability = [98.2, 95.4, 87.3, 71.2, 48.6, 18.3, 4.2] ↑ high ↑ low decreasing → viability[::-1] = [4.2, 18.3, 48.6, 71.2, 87.3, 95.4, 98.2] ↑ low ↑ high increasing → np.interp() requires values to be INCREASING so we must reverse viability first

💻 Code — day3_ic50_calculation.py

import numpy as np
import matplotlib.pyplot as plt
np.set_printoptions(suppress=True)

concentrations = np.array([0.1, 0.3, 1.0, 3.0, 10.0, 30.0, 100.0])
viability      = np.array([98.2, 95.4, 87.3, 71.2, 48.6, 18.3, 4.2])
log_conc       = np.log10(concentrations)

# Step 1: reverse arrays (np.interp needs increasing values)
viability_flipped = viability[::-1]
log_conc_flipped  = log_conc[::-1]

# Step 2: interpolate to find log(IC50)
log_ic50 = np.interp(50, viability_flipped, log_conc_flipped)
# np.interp(target_y, y_values, x_values)
# "At what x does y equal 50?"

# Step 3: convert log back to nM
ic50 = 10 ** log_ic50
# 10 ** log = antilog = converts back to concentration
# 10**1 = 10, 10**1.5 = 31.6, 10**2 = 100

print(f"IC50 = {ic50:.2f} nM")
print(f"Log IC50 = {log_ic50:.3f}")

# Plot with IC50 marked
plt.figure(figsize=(8, 5))
plt.plot(log_conc, viability, 'o-', color='steelblue',
         linewidth=2, markersize=8)
plt.axhline(y=50, color='red', linestyle='--',
            alpha=0.7, label='50% viability')
plt.axvline(x=log_ic50, color='green', linestyle='--',
            alpha=0.7, label=f'IC50 = {ic50:.1f} nM')
# plt.axvline() = vertical line at x position

plt.xlabel('Log10 Concentration (nM)')
plt.ylabel('Cell Viability (%)')
plt.title('IC50 Calculation')
plt.legend()
plt.grid(True, alpha=0.3)
plt.ylim(0, 110)
plt.tight_layout()
plt.savefig('ic50_plot.png', dpi=150)
plt.show()

✅ Expected Output

IC50 = 9.24 nM Log IC50 = 0.966

📅 Stage 2 — Day 4 — Thursday

Multi-Compound Screening

⏱ 40–50 min 🆕 dictionaries, .items(), enumerate()
🔬 Scenario
Your lab tested three conditions — chemotherapy alone, a RAGE inhibitor alone, and their combination — in the same dose-response assay. You need to rank all three by IC50 and plot them on one graph to show the combination benefit.

💡 New Concept: Dictionaries

A dictionary stores key:value pairs: compounds = { 'Drug A': np.array([98.2, 87.3, 48.6, 4.2]), 'Drug B': np.array([99.1, 91.4, 61.2, 12.1]), } Access by key: compounds['Drug A'] Loop through: for name, viability in compounds.items() → name = 'Drug A' → viability = the array

💻 Code — day4_compound_screening.py

import numpy as np
import matplotlib.pyplot as plt
np.set_printoptions(suppress=True)

concentrations = np.array([0.1, 0.3, 1.0, 3.0, 10.0, 30.0, 100.0])
log_conc       = np.log10(concentrations)

compounds = {
    'Doxorubicin':  np.array([98.2, 95.4, 87.3, 71.2, 48.6, 18.3,  4.2]),
    'TTP488':       np.array([99.2, 97.8, 94.1, 83.4, 62.7, 38.2, 15.4]),
    'Dox + TTP488': np.array([97.1, 91.3, 78.4, 54.2, 28.7,  9.1,  2.3]),
}
colors = ['steelblue', 'darkorange', 'green']

# Calculate IC50 for each compound
ic50_results = []

for compound_name, viability in compounds.items():
    viab_flip    = viability[::-1]
    log_c_flip   = log_conc[::-1]
    log_ic50     = np.interp(50, viab_flip, log_c_flip)
    ic50         = 10 ** log_ic50
    ic50_results.append((compound_name, ic50))

# Sort by IC50 — lowest = most potent
ic50_results.sort(key=lambda x: x[1])

print(f"{'Compound':<20} {'IC50 (nM)':>10} {'Rank':>6}")
print("-" * 40)
for rank, (name, ic50) in enumerate(ic50_results, start=1):
    print(f"{name:<20} {ic50:>9.2f}  {rank:>5}")

# Plot all curves
plt.figure(figsize=(9, 6))
for (name, viability), color in zip(compounds.items(), colors):
    viab_flip  = viability[::-1]
    log_c_flip = log_conc[::-1]
    log_ic50   = np.interp(50, viab_flip, log_c_flip)
    ic50       = 10 ** log_ic50
    plt.plot(log_conc, viability, 'o-', color=color,
             linewidth=2, markersize=7,
             label=f'{name} (IC50={ic50:.1f} nM)')

plt.axhline(y=50, color='red', linestyle='--', alpha=0.5)
plt.xlabel('Log10 Concentration (nM)')
plt.ylabel('Cell Viability (%)')
plt.title('Combination Screen — TNBC Cells')
plt.legend(fontsize=9)
plt.grid(True, alpha=0.3)
plt.ylim(0, 110)
plt.tight_layout()
plt.savefig('compound_screen.png', dpi=150)
plt.show()

✅ Expected Output

Compound IC50 (nM) Rank ---------------------------------------- Dox + TTP488 3.12 1 Doxorubicin 9.24 2 TTP488 29.41 3
