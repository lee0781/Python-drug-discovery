# Python for Drug Discovery — Learning Portfolio

A self-directed portfolio applying Python to real drug discovery 
workflows, anchored to publicly available assay data and 
biotech research methods.

## Roadmap

**Stage 1 — NumPy Fundamentals** ✅ Complete
Arrays, statistics, 2D plate simulation, Boolean masking applied 
to cell viability assays and plate reader data.
`numpy_assay_basics.py`


**Stage 2 — Dose-Response Curves + IC50** ← In Progress
Plotting drug-response relationships and fitting curves using 
publicly available osimertinib (AstraZeneca) EGFR inhibitor data.

**Stage 3 — Pandas + HTS Data Tables** (upcoming)
DataFrames, CSV pipelines, QC filtering on publicly available 
HTS datasets including venetoclax (AbbVie) BCL-2 inhibitor screen.

**Stage 4 — SciPy + CRISPR Analysis** (upcoming)
4-parameter logistic curve fitting, CRISPR dropout screens, 
volcano plots using Broad Institute published methodology.

**Stage 5 — Machine Learning** (upcoming)
Drug sensitivity prediction and ML on cancer cell line panels 
using scikit-learn.
